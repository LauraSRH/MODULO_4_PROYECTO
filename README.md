# MODULO_4_PROYECTO.  Buscador de Pokémon con PokéAPI
Archivo con el script del programa con el uso de la API porkeapi.com.
El programa se inicia con la importación de request, json, webbrowser, os y pathlib. Posteriormente, con ayuda de la Inteligencia Artificial de VSCode se desarrollo el programa con funciones para obtener los datos del pokemon, con try para conectar con la url, if para el status de respuesta, con sus excepciones para algunos errores de conexión o inesperados.

Otra función para procesar los datos del Pokémon, como movimientos, habilidades, tipo e imagen. La función para mostrar la información o datos del Pokémon. La función para guardar los datos en el archivo pokédex. Otra función para mostrar la imagen, y la última función para organizar el programa. 

Todas las funciones usaron comandos como ciclo for, try, if, with, else y return dependiendo de la función.

El programa se corrió con éxito, ingresándose nombres de varios Pokemones, como Pikachu, Articuno, Regirock, Silvally y Dializa, en los que la API retorno los datos y con Codigo Cero donde retorno el código 400.

De forma adicional, para darle mejor terminación, se agregaron dos funciones adicionales, una que pregunta si se quiere buscar otro Pokémon, indicando con s para si y n para no, y otra que da la opción de enter para cerrar el programa.

Para verificar todo el programa se revisaron las funciones y el scrip o códigos, para que cumpliera con los requisitos establecidos en el proyecto.
